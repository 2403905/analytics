# Analytics

The Analytics is a CLI application that has the outputs:

[1] Top 10 active users sorted by amount of PRs created and commits pushed  
[2] Top 10 repositories sorted by amount of commits pushed  
[3] Top 10 repositories sorted by amount of watch events

### Installation
* Clone a repository
* Call `make` command to run an application  
Notice: The `make` command call the wget and tar tools to set up the required dependencies. If you don't have such tools please install it or downloading required dependencies manually.
